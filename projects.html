multichat: simulated chat application across a network

Java

multichat consists of 2 different kinds of programs: a server and a client. Multiple instances of the client program can run at the same time and messages can be sent between clients and the server. New clients can join and leave the group at any point in time. I am also working on message delivery guarantee, so that one party can guarantee the message it sent had been received. The main purpose of this project is to practice working with multithreading and networking in Java.
